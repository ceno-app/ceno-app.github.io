تبدیل به یک پل شوید

افرادی که در کشورها یا مناطقی با دسترسی آزاد و پایدار به اینترنت ساکن‌اند و از سِنو استفاده می‌کنند، می‌توانند با تبدیل شدن به یک «پل» به کسانی که در شبکه‌های به‌شدت محدود قرار دارند کمک کنند تا به اینترنت دسترسی داشته باشند. اجرای پل اقدامی بسیار ساده و در عین حال قدرتمند است که به شما امکان می‌دهد محتوای اینترنت را با دیگران به اشتراک بگذارید.

درباره چگونگی تبدیل شدن به پل بیشتر بدانید

The simple way:

Run Ceno Browser on your Android device, and/or on a spare, always-on device.

Run the Ceno Docker Client:

  1. With just one command you can set up a bridge on your computer or VPS:
    sudo docker run --name ceno-client -dv ceno:/var/opt/ouinet --network host --restart unless-stopped equalitie/ceno-client
  2. Open a portal to the internet: Ensure UPnP is enabled on your router, or set up port forwarding.
  3. To see if it's working, create and open a new Firefox Profile, and set the proxy to localhost, port 8077.
  4. Install the Ceno Web Extension and the provided certificate to test it out!
