3. Повідомлення про конфіденційність

Цей розділ використовується для інформування відвідувачів про політику eQualitie щодо збору, використання та розкриття особистої інформації для всіх користувачів цієї служби.

3.1 Збір даних

Ceno Browser is designed with privacy at its core and does not require any personally identifying information to function. You do not need to provide a name, user ID, nickname, account, or any other personal data to use the app. eQualitie does not know who uses Ceno, nor with whom users synchronize or share content.

We value your privacy and only collect anonymized usage data to help improve the application. This information is stripped of any personally identifiable elements before collection and is never shared with third parties.

By default, Ceno collects anonymized usage metrics using our own, self-hosted, analytics server. This process is designed to be privacy-preserving:

  • IP addresses are not stored
  • No personal information is collected
  • Data is associated with a per-device randomly generated ID
  • The randomly generated ID is not associated with any user or device identifiers (e.g. IP, MAC, IMEI, MEID)
  • The ID is replaced with a new random ID every 30 days or less
  • Past, current and future random device IDs are not associated

The following anonymized data may be collected:

  • Peer to peer network statistics, e.g. data volume transferred, count of peer connections made
  • Application performance data, e.g. connection success and latency
  • Application usage statistics, e.g. time spent in public vs. personal browsing modes
  • Country where device is located (Inferred. No GPS or specific location is sent)
  • Device settings relating to country, language, and locale
  • Whether the device is using a VPN (but no further information about the VPN)
  • The cellular service provider name (if applicable)

Note: country inference is based on device locale, VPN presence, and cellular provider data.

All collected data is automatically deleted after 90 days. Users may opt out of metrics collection at any time via the app settings.

Ми збираємо з погодинними, щоденними та місячними інтервалами різні сукупні вимірювання, щоб допомогти нам зрозуміти кількість користувачів, включно з джерелом ASN мережі та країною, на основі перегляду в загальнодоступному режимі.

Injectors are servers that can see requests for content on the network, including the IP address of the user making the request and the name of the requested website domain. The injectors are a necessary network component for 1) introducing web content into a decentralized network, and 2) authenticating the integrity of introduced content. On the Ceno network, Injectors are run and managed by the Ceno team. eQualitie deletes all access logs every 24 hours.

Сервери Bootstrap — це сервери, які використовуються для введення вашого пристрою в мережу BitTorrent. Ми надсилаємо IP-адреси користувачів як на наші власні сервери завантаження, так і на сторонні сервери завантаження. Для саморозміщених серверів початкового завантаження ми зберігаємо IP-адреси в кеш-пам’яті. IP-адреси час від часу перевіряються на активність і видаляються, коли вони перестають реагувати на трафік BitTorrent.

3.2 Обмін даними

Браузер Ceno (і eQualitie) не передає жодних даних третім особам.

The application offers two modes of operation - Public and Personal.

Public tabs use all connectivity methods available in the Ceno network to get the desired content. When browsing using a Public tab, users will leave a record of their activity on the public BitTorrent network (IP address and website requested). Content that has been retrieved by a user's device inside a Public tab will be shared with other Ceno users via the BitTorrent network.

У режимі Персональний ви зв'яжетеся з Інжекторами лише для того, щоб вони знайшли і доставили вам цей веб-сайт. Пошук не підключатиметься до мережі BitTorrent і не намагатиметься знайти вміст на пристроях інших користувачів.

To ensure that your Ceno client can always contact an Injector, we have also created Bridges. If the Injectors are blocked on your network, the Ceno app will look for available Bridges, who will forward a user's request to the Injectors. Users may opt in to enabling Bridge Mode in Ceno Browser to improve web content accessibility for others. Bridge activity is invisible to the Bridge operator (unless they are actively monitoring their device for incoming and outgoing connections). Their job is to receive network requests from other Ceno users for access to an Injector and forward this request. Bridge operators will have their device's IP address listed in the BitTorrent network as an accessible Bridge.

3.3 Дані журналу

Користувачі можуть увімкнути реєстрацію через меню налаштувань у Ceno Browser. Якщо увімкнено, Ceno Browser створює лог-файли на пристроях користувачів. Метою цих файлів є лише реєстрація активності пристрою для полегшення процесу налагодження у випадку, якщо користувач відчуває труднощі при з'єднанні з іншими користувачами або при використанні Ceno іншим чином. Лог-файл залишається на пристрої користувача, якщо тільки користувач не вирішить надіслати його нам з метою підтримки, і автоматично видаляється, коли його розмір досягає 15 МБ.

Якщо користувач все-таки вирішить зв’язатися з нами, персональні дані, які ми можемо збирати, це:

  • Адреса електронної пошти - якщо користувач вирішив зв'язатися з нами електронною поштою;
  • Інформація, яку користувач може надати електронною поштою, через довідкові квитки або через веб-сайти eQualitie, а також пов’язані метадані – з метою надання технічної підтримки;
  • Діяльність веб-перегляду та історія.

Жодна з цих даних не передається третім особам.

3.4 Звіт про збої

Users can opt-in to crash reporting via the Settings menu in Ceno Browser. When enabled, Ceno Browser will send crash reports in the event of an issue while running the app. Crash reports are forwarded automatically to a server run by eQualitie. We use Sentry for crash reporting.

Sentry зберігає дані про події у виробництві протягом 90 днів. Окремі події об’єднуються в ширші проблеми та видаляються після зазначеного вище періоду зберігання. Проблеми видаляються, коли видаляються всі пов’язані події.

Сервери Bootstrap: сервери, які допомагають вашому пристрою підключитися до мережі BitTorrent.

Мости: сервери (включно з пристроями користувачів, якщо ввімкнено режим мосту), на яких працює браузер Ceno, які служать для маршрутизації запитів веб-вмісту, якщо інші механізми підключення недоступні або недоступні.

Інжектори: Сервери, керовані eQualitie, які спрямовують веб-контент з серверів походження на пристрої користувачів Ceno, що запитують цей контент.

Особистий режим. Діяльність веб-перегляду в особистому режимі нікому не надається. Веб-вміст безпечно отримується з джерела або через Injectors.

Публічний режим: Під час перегляду в публічному режимі використовуються всі доступні методи підключення для отримання веб-вмісту з мережі Ceno, обмін цим вмістом з іншими користувачами Ceno через мережу BitTorrent, а також публічний запис про IP-адресу вашого пристрою та отриманий вміст.
