Resilient

Les sites Web sont partagés par un réseau mondial de pairs, et stockés dans un cache distribué pour la disponibilité lorsque les réseaux traditionnels sont bloqués ou descendent.

Coût des données

Vous payez des coûts de données plus faibles parce que Ceno Browser achemine le trafic d'utilisateurs par le biais de réseaux entre pairs, tout en vous fournissant encore une capacité de contournement.

Gratuit et open source

Ceno Browser est propulsé par une bibliothèque open source appelée Ouinet qui permet aux développeurs tiers d'intégrer le réseau Ceno dans leurs applications pour la connectivité entre pairs.
