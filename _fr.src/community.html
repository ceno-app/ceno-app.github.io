Construisons un meilleur internet ensemble

Devenez pont

Les gens courent Ceno de pays avec accès libre à Internet, ou de parties du monde où l'accès à Internet est fiable, peut aider ceux dans les réseaux fortement restreints à accéder à Internet en devenant un "Bridge". Courir un pont est une action très simple et puissante, vous permettant de partager le contenu d'Internet avec d'autres.

En savoir plus sur le fait de devenir un pont.

La manière simple:

Lancez Ceno Browser sur votre appareil Android, et/ou sur un appareil de rechange, toujours sur.

Courez le client Ceno Docker:

  1. Avec une seule commande vous pouvez configurer un pont sur votre ordinateur ou VPS:
    sudo docker run --name ceno-client -dv ceno:/var/opt/ouinet --network host --restart unless-stopped equalitie/ceno-client
  2. Ouvrez un portail sur Internet : Assurez-vous que UPnP est activé sur votre routeur, ou configurez l'expédition portuaire.
  3. Pour voir s'il fonctionne, créer et ouvrir un nouveau profil Firefox, et définir le proxy à localhost, port 8077.
  4. Install the Ceno Web Extension and the provided certificate to test it out!
