Qu’est-ce que le Navigateur Ceno ?

Ceno est un navigateur web mobile basé sur Mozilla Android Components. Il est conçu pour l'accès et la distribution de contenu Web (websites) en utilisant une nouvelle approche qui repose sur BitTorrent – un réseau décentralisé de partage de fichiers. L'avantage de Ceno provient des méthodes de censure Internet en cours et permet aux gens d'accéder et de partager des informations Web dans et à travers les régions où la connectivité a été interrompue ou compromise.

Comment est ce que Ceno fonctionne ?

Lorsqu'un site web est accessible à partir de votre réseau (par exemple, en accédant à www.canada.ca à partir de votre Internet domestique), Ceno s'y connectera comme un navigateur Web normal.

Lorsqu'un site Web n'est pas accessible en raison d'un filtrage/censure du réseau, Ceno tentera d'obtenir le contenu pour vous. La manière dont Ceno tente de trouver du contenu dépend de si vous avez opté pour utiliser Ceno en mode Public ou Personnel.

Dans le mode Public, Ceno se penchera sur le réseau BitTorrent pour voir si un autre utilisateur Ceno a récemment partagé la page demandée. Si le service peut identifier la page demandée, il récupérera cette page à partir du périphérique d'un autre utilisateur. Si le contenu n'est pas disponible, Ceno contactera plusieurs Injecteurs pour demander ce site Web et l'avoir livré à vous.

Dans le mode Personnel, vous ne contacterez les Injecteurs que pour que ce site soit fétiché et livré à vous. La recherche ne se connectera pas au réseau BitTorrent et ne tentera pas de localiser le contenu sur les appareils d'autres utilisateurs.

Pour garantir que votre client Ceno puisse toujours contacter un injecteur, nous avons également créé des ponts. Si les injecteurs sont bloqués sur votre réseau, l'application Ceno recherchera les ponts disponibles, qui transmettront votre demande aux injecteurs. Le réseau Ceno compte actuellement environ 6 000 ponts. Leur nombre ne cesse de croître.

Quels sont les avantages d’utiliser Ceno ?

Notre devise est ‘Partager le Web!’ et c’est essentiellement ce que font les utilisateurs de Ceno. La plupart des pages Web ont un contenu statique (quelque chose vous lisez, regardez, écoutez, etc.). Le contenu statique peut être emballé et partagé avec d'autres utilisateurs recherchant le même contenu. Ceno utilise le réseau de partage de fichiers BitTorrent pour cela.

L'avantage de Ceno est évident dans un réseau qui censure les sites Web, ou où le réseau est faible ou instable. Dans ces circonstances, le client Ceno peut utiliser des méthodes supplémentaires pour récupérer le contenu et le livrer à votre appareil. En permettant aux utilisateurs de permettre le partage de contenu qu'ils ont récemment récupérés, Ceno élargit l'accès à Internet plus généralement. Que vous vivez dans une partie du monde qui limite l'accès au contenu, ou dans une zone éloignée avec un accès limité à Internet, Ceno vous permet d'accéder à des informations qui peuvent être autrement indisponibles.

Les personnes qui dirigent Ceno depuis des pays qui ne censurent pas Internet, ou depuis des régions du monde où l’accès à Internet est fiable, peuvent choisir de servir de ponts pour ceux qui utilisent des réseaux fortement restreints. Le réseau inscrit automatiquement les nouveaux ponts et assure le suivi de leur mise hors ligne. Lancer un pont est une action très simple, mais puissante, vous permettant d'aider les autres à accéder à Internet.

Ceno - and all accompanying components of the tool that together create a decentralized network of peers and nodes - is free and open source software. You can inspect our code or re-create the entire network yourself!

Quels sont les risques liés à l’utilisation de Ceno ?

Risques pour les utilisateurs

Chaque technologie traitant de cas d’utilisation sensibles présente ses avantages et ses risques. De par sa conception, Ceno utilise le réseau public BitTorrent pour une grande partie de ses besoins de routage et de stockage de données. Chaque fois que vous demandez ou partagez un élément de contenu sur le réseau BitTorrent (mode public), vous créez un enregistrement qui associe votre adresse IP à un élément de contenu récupéré d'un site Web particulier. Par exemple, n'importe qui peut interroger le réseau BitTorrent pour obtenir une liste d'adresses IP associées à website.com.

Les utilisateurs partageant le contenu récupéré avec d'autres peuvent également encourir des frais de données supplémentaires lorsque le contenu est copié de leur appareil vers l'appareil d'autres utilisateurs de Ceno.

En réponse à ces risques, nous avons créé un mode "Personal" pour les utilisateurs de Ceno où le service n'interagit pas avec le réseau BitTorrent. Nous recommandons ce mode pour les utilisateurs qui ne veulent aucun enregistrement de leur activité à apparaître sur BitTorrent, ainsi que pour les opérations de site Web qui nécessitent une authentification (par exemple lorsque vous vous connectez à vos comptes de médias sociaux).

Vous pouvez facilement basculer entre les onglets Public et Personnel de la même manière que vous basculez entre les onglets Normal et Privé/Incognito sur votre navigateur actuel.

Les injecteurs peuvent également voir les demandes de contenu sur le réseau, y compris l'adresse IP de l'utilisateur qui fait la demande et le nom du site web proposant le contenu souhaité. Les injecteurs sont un composant réseau nécessaire pour 1) introduire du contenu Web dans un réseau décentralisé et 2) authentifier l'intégrité du contenu introduit. Sur le réseau Ceno, les injecteurs sont gérés et gérés par l'équipe Ceno. Nous supprimons périodiquement tous les journaux d'accès.

Risques pour les opérateurs de ponts

Puisque n’importe quel utilisateur peut devenir un pont, considérez les risques introduits par cette action.

L'activité du pont est invisible pour l'opérateur du pont (à moins qu'il surveille activement son appareil pour les connexions entrantes et sortantes). Leur travail consiste à recevoir les demandes réseau des autres utilisateurs de Ceno pour accéder à un injecteur et à transmettre cette demande.

Les exploitants de ponts auront l'adresse IP de leur appareil indiquée dans le réseau BitTorrent en tant que pont accessible.

Le pont ne fonctionne que lorsque le réseau dans lequel vous vous trouvez autorise les connexions entrantes. Ceci est vérifié automatiquement par l'application client Ceno au démarrage. La plupart des opérateurs de téléphonie mobile sans fil n'autorisent pas les connexions entrantes.

Qui développe le navigateur Ceno ?

Ceno is a technology created and maintained by the Canadian not-for-profit organization eQualitie. They have incorporated their Ouinet network library into the Firefox Android browser.

eQualitie existe depuis douze ans, période pendant laquelle ils ont créé et développé des outils pour protéger la liberté d'expression, faciliter l'échange d'informations et promouvoir l'expression de soi sur Internet.

Certains de leurs autres projets incluent Ouisync, un service de partage de fichiers décentralisé ; Deflect, un outil de protection DDoS ; Baskerville, un système de détection d'attaques basé sur l'apprentissage automatique ; et la boîte à outils de déploiement dComms pour la mise en place de services réseau décentralisés et fédérés.

Quelle est la différence entre l'utilisation d'un VPN et Ceno Browser ?

Les VPN ont été initialement créés pour les communications privées entre votre appareil et un serveur de destination. Alors que la censure d’Internet est devenue fréquente dans le monde entier, les fournisseurs de VPN ont commencé à proposer leurs services afin de contourner également les restrictions des réseaux locaux. Aujourd’hui, il existe de nombreux fournisseurs de VPN proposant une grande variété d’options de connectivité pour aider leurs utilisateurs à contourner la censure des réseaux locaux.

Les VPN peuvent voir qui sont leurs clients et quels sites web ils demandent. Ceci est inhérent à la conception et un niveau de confiance dans l'opérateur VPN est requis (un peu comme la confiance requise dans eQualitie, qui exploite les injecteurs Ceno).

La principale distinction de Ceno d'un VPN est qu'il s'efforce d'aligner toutes les demandes de votre site Web via le réseau décentralisé. Lorsqu'un site Web est disponible sans restriction, Ceno se connecte simplement à lui comme un navigateur Web normal. En outre, les utilisateurs de Ceno cachent et partagent du contenu l'un avec l'autre. Cela réduit la pression sur les noeuds de contournement de la censure et améliore la délivrance.

Quelle est la différence entre l'utilisation des navigateurs Tor et Ceno?

Contrairement à Tor Browser, Ceno Browser n'est pas un outil pour l'anonymat, qui est le but principal de Tor. Dans le réseau Tor, le trafic réseau est crypté et acheminé par un réseau de relais gérés par des bénévoles et semble provenir de l'adresse IP d'un nœud de sortie. Tor est une excellente option pour la protection de la vie privée des opérateurs Internet et de site Web. Si elle fonctionne dans votre environnement réseau, nous le recommandons, à condition que vous ayez également lu leur documentation de support.

Quelle est la différence entre les onglets Personnel et Public, encore une fois ?

Les onglets publics utilisent toutes les méthodes de connectivité disponibles sur le réseau Ceno pour obtenir le contenu souhaité. Lorsque vous naviguez à l'aide d'un onglet Public, vous laisserez une trace de votre activité sur le réseau public BitTorrent (votre adresse IP et le site web que vous avez demandé). Le contenu récupéré par votre appareil dans un onglet Public sera partagé avec d'autres utilisateurs de Ceno via le réseau BitTorrent.

Les informations demandées et récupérées dans un onglet Personnel ne sont partagées avec personne d’autre. Aucune trace de votre activité n'est enregistrée sur BitTorrent. Utilisez des onglets Personnel pour vous connecter aux réseaux sociaux et à d’autres comptes. Utilisez-les également si vous ne souhaitez pas être associé sur BitTorrent à aucune de vos activités de navigation dans Ceno.
